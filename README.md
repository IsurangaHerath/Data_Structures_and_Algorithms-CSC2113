# Data_Structures_and_Algorithms-CSC2113

**Data Structure: A way to store and organize data for efficient use.
Algorithm: A step-by-step procedure to solve a problem.
In Java, DSA is implemented using classes, objects, arrays, and built-in collections.**

## 1. Stack (LIFO)
**Concept:** Last-In-First-Out (LIFO) – the last element added is removed first.  

**Operations:**  
- `push()` – add element  
- `pop()` – remove element  
- `peek()` – view top element
